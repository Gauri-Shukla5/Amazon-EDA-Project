# Amazon EDA Project

## Overview
This project performs Exploratory Data Analysis (EDA) on Amazon product dataset.

## Objectives
- Analyze product categories
- Study price vs rating relationship
- Identify top-selling products
- Understand customer behavior

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- High-priced products do not always have high ratings
- Some categories dominate the market
- Customer reviews influence product popularity
